# KLauncher
KLauncher is a launcher for Minecraft: Legacy Edition, made by me, with the Godot Engine.

## Main Menu
<img width="1920" height="1079" alt="image" src="https://github.com/user-attachments/assets/408685bd-1406-4ed6-b976-993fa55029dd" />

It features a fully functioning options menu, allowing you to configure the fork, fork name, your username, UID, and more.

Furthermore it allows you to check for updates to easily check your instance for any updates that may be available, it does this by downloading the Minecraft.Client.exe file (or whatever you have to set it up to download) and checking it's checksum against the previously downloaded checksum.
